# Boolean-Expressions
Example demonstrates different boolean expressions in Go like ||,&&,!. Click [here](https://tour.golang.org/basics/11) to learn more

```go
func main() {
	fmt.Println("true && false = ", (true && false))
	fmt.Println("true || false = ", (true || false))
	fmt.Println("!true = ", !true)
}
```

### Output

```bash
true && false =  false
true || false =  true
!true =  false
```
