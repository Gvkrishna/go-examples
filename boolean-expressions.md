# Boolean-Expressions
Example demonstrates different boolean expressions in Go like ||,&&,!

```go
func main() {
	fmt.Println("true && false = ", (true && false))
	fmt.Println("true || false = ", (true || false))
	fmt.Println("!true = ", !true)
}
```

```bash
true || false =  true
!true =  false
```
